### Windows
- Create venv and install requirements
    ```shell
    python -m venv venv
    venv\Scripts\activate
    pip install -r requirements.txt
    ```
- run script
    ```shell
    python run.py
    ```
- Open [http://127.0.0.1:25000/]() to test work :D

### MacOS/Linux
- Create venv and install requirements
    ```shell
    python3 -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
    ```
- run script
    ```shell
    python3 run.py
    ```
- Open [http://127.0.0.1:25000/]() to test work :D
